
# Edn

A simple utility to pretty-print [Edn](https://github.com/edn-format/edn) formatted data.

Edn parsing library used and modified from [hedn](https://bitbucket.org/dpwiz/hedn).

## Usage

		> cabal install
		> echo '{:test true}' | edn
